# ionic_appoint

ionic版本appoint

# 开发相关

## 相关命令

* gulp watch

	开发时使用，用于监控js/css等文件的压缩合并。

* gulp minify

	构建前使用，生成合并之后的文件。

* ionic serve --lab

	开发时使用，用于在浏览器中进行调试。

* npm install

	安装插件。

* ionic platform add ios[android]

	添加对应的平台。

* ionic build

	平台打包。

# 注意事项

* www/js/build目录中文件需要手动生成。参考gulp minify命令。
